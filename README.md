### 功能缺陷及优化

在提交完注册后，前端浏览器会等待后端发送完激活邮箱的响应，此时会在前端页面导致四到五秒的等待时间，之后才会跳转到提示激活的页面。其实这段时间是完全不用用户去等待的，在用户提交完注册之后，其实就可以直接跳转到提示注册的页面。

解决方案是使用异步技术，分别由不同的线程来解决等待响应的问题。
